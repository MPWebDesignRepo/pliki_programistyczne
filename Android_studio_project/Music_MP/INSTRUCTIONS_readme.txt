ENG:
The Music_MP_WebDesign application (located in the folder /Android_studio_project/Music_MP) is created 
in the Dart programming language using the Flutter framework 
and Android Studio.

Requirements for running it:

Access to the application files (download)
Running an installed and configured Android Studio
Opening the Music_MP application folder
Terminal (enter the command flutter run)
After entering the command in the Android Studio terminal, if the emulator does not start, you can choose a web browser in which the application should be displayed (options 1, 2, 3).



In the folder, you will find all the steps for creating this application based on training materials.

By running it in the /step_07 or /final folder, you will launch the completed application.
